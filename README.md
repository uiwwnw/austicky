## How to use

```javascript

var a = new auSticky('header');

// multi
var a = new auSticky('header, nav, .ac');

// option
var a = new auSticky('footer', {
    activeClassName: 'aaa', // edit sticky active class name
    bottom: true // trun on bottom type 
});

```
